# Sales Analytics Dashboard

A complete EDA and interactive dashboard for Superstore Sales data.

## 📋 Project Overview

**Dataset:** Superstore Sales (134,000+ rows)

**Goals:**
1. Perform full EDA on the sales dataset
2. Engineer new features (month, quarter, profit margin)
3. Build 10+ visualizations
4. Find 5 actionable business insights
5. Deploy as an interactive Streamlit dashboard

## 🚀 Getting Started

```bash
pip install pandas matplotlib seaborn plotly streamlit
streamlit run dashboard.py
```

## 📁 Files

| File | Description |
|------|-------------|
| `eda.ipynb` | Full EDA notebook |
| `dashboard.py` | Streamlit dashboard |
| `data/superstore.csv` | Dataset (download from Kaggle) |

## 💡 Business Questions to Answer

1. Which product categories generate the most profit?
2. Which regions have the highest sales but lowest profit? (leaking margin)
3. What is the relationship between discount and profit?
4. Which customer segments are most valuable?
5. Are there seasonal patterns in sales?
