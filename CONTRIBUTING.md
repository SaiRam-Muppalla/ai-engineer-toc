# Contributing to AI Engineer Curriculum

Thank you for wanting to contribute! This project thrives on community contributions.

## 🤝 Ways to Contribute

1. **Fix bugs** — Broken code, outdated links, typos
2. **Add projects** — New real-world AI project templates
3. **Add tutorials** — Step-by-step guides for key concepts
4. **Add exercises** — Practice notebooks
5. **Add resources** — Useful courses, videos, books
6. **Share your portfolio** — Add your completed project to the showcase

## 🚀 How to Contribute

1. Fork this repository
2. Create a branch: `git checkout -b feature/add-new-project`
3. Make your changes
4. Test that all code runs correctly
5. Open a Pull Request with a clear description

## 📋 Contribution Standards

### For Code/Notebooks
- Python 3.10+ compatible
- Clear comments and docstrings
- Includes a brief README in the project folder
- Tested and runs without errors

### For Resources
- Must be free or have a free tier
- Must be in English or have English subtitles
- Must be relevant to the stage it belongs in
- Provide a one-line description

### For Projects
- Include a `README.md` with: goal, tech stack, instructions, expected output
- Starter code with clear TODOs
- No proprietary datasets (use publicly available data)

## 💬 Questions?

Open an issue if you have questions about contributing.
