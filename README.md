## 📚 Curriculum Stages

### [Stage 0 — Foundations](./00-foundations/)
Math and programming fundamentals required for AI. Linear algebra, calculus, statistics, and probability.

### [Stage 1 — Python for AI](./01-python-for-ai/)
Python mastery with a focus on AI use cases: OOP, file I/O, APIs, NumPy, and Pandas.

### [Stage 2 — Data Analysis](./02-data-analysis/)
Data wrangling, EDA, feature engineering, and data visualization with real datasets.

### [Stage 3 — Machine Learning](./03-machine-learning/)
Supervised, unsupervised, and reinforcement learning fundamentals with Scikit-learn.

### [Stage 4 — Deep Learning](./04-deep-learning/)
Neural networks, CNNs, RNNs, Transformers, and practical model training with PyTorch.

### [Stage 5 — LLM Engineering](./05-llm-engineering/)
Prompt engineering, RAG systems, vector databases, fine-tuning, AI agents, and multi-agent systems.

### [Stage 6 — MLOps](./06-mlops/)
Model serving, FastAPI, Docker, CI/CD, experiment tracking, monitoring, and cloud deployment.

### [Stage 7 — AI Projects](./07-ai-projects/)
15+ end-to-end real-world AI projects with code, explanations, and deployment guides.

### [Stage 8 — AI Products](./08-ai-products/)
Building AI-powered products and startups: product design, monetization, and user acquisition.

### [Stage 9 — Career Preparation](./09-career-preparation/)
Portfolio building, AI engineer interview prep, system design, open source contributions.
